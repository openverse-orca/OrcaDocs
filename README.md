# 欢迎使用 OrcaLab ！

欢迎使用 OrcaLab 系列工具的官方文档。本手册将指导你完成从软件安装、用户注册到功能使用的全流程，涵盖快速入门仿真案例、操作指南、开发指南等核心内容。

## 产品定位 
OrcaLab 是松应科技推出的 高保真物理AI仿真系统轻量版，聚焦 “告别复杂代码，人人可学的机器人训练术” 核心理念，专为高校师生、个人用户及非商用场景设计，旨在通过轻量化功能布局降低机器人虚拟仿真训练的使用门槛，同时作为 OrcaStudio 企业版的推广载体，让用户快速体验机器人训练全流程核心能力。 

---
## 核心优势 
- **易用性强**：无需复杂代码基础，提供模块化、可视化操作流程，适配高校教学、个人学习等非专业开发场景，快速上手机器人虚拟仿真训练。 
- **全流程覆盖**：从物理 AI 仿真环境创建、数据采集，到数据增广、清洗、模型训练，再到真机能力迁移，完整覆盖机器人虚拟仿真训练关键环节。 
- **灵活适配**：支持 3D AI 场景生成、自定义数据处理规则、多样化模型训练等个性化操作，满足不同用户的实验、学习需求。  

## 适用人群 
- **高校师生**: 用于机器人相关课程教学、实验操作、学术研究等场景。
- **个人用户**：非商用目的的机器人训练技术学习、兴趣探索、技能实践等场景。 
- **潜在企业用户**：希望通过轻量化版本提前体验 Orca 系列产品核心能力，为后续商用订阅决策提供参考。  

## 产品形式 
独立安装软件，支持一键快速部署，人人可学的机器人训练术，让智能为你所用。
