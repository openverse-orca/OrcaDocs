# OrcaLab 用户注册
欢迎加入`OrcaLab`！注册账号是开启专属服务、解锁完整功能的关键一步，让我们为你带来安全、个性化的使用体验。


## 一、用户注册指南

### 1.1 用户注册与登录
打开资产库页面https://simassets.orca3d.cn/ ，已注册用户直接登录，未注册用户需完成注册后使用。

 **方式一：手机/邮箱注册账号**：

   - 点击注册按钮
   - 输入手机号/邮箱地址（会收到验证码）
   - 输入验证码   
   - 填写注册信息：
     - First Name（名）
     - Last Name（姓）
     - 用户名
     - 手机号
     - 密码（输入两次）
     - 填写邀请码
   - 勾选阅读并接受（用户协议，隐私政策）      
   - 提交注册

 **方式二：第三方账号注册**：  支持微信和github第三方账号注册
   - 点击第三方注册选项
     - 根据提示微信扫码或者输入github账号邮箱
   - 进入下一步绑定邮箱
   - 输入验证码
   - 填写注册信息：
     - First Name（名）
     - Last Name（姓）
     - 用户名
     - 手机号
     - 密码（输入两次）
     - 填写邀请码
   - 勾选阅读并接受      
   - 提交注册
   
### 1.2 注册成功自动登录
注册成功后会自动登录，并弹出资产库界面

## 二、用户注册常见问题
#### 2.1 邮箱验证码问题

**⚠️ Gmail 邮箱已知问题**：

如果使用 Gmail 邮箱注册，可能会遇到以下问题：

```
收件服务器拒收您的邮件可能是由于您的SPF验证不通过
```

**原因**：从 2022 年 11 月起，谷歌要求向 Gmail 邮箱发送邮件的发件人必须设置 SPF 或 DKIM 记录。

**解决方案**：

- 使用非 Gmail 邮箱注册（推荐）
- 或联系系统管理员配置 SPF/DKIM 记录

**SPF 记录示例**：

```
v=spf1 include:spf.163.com -all
```
